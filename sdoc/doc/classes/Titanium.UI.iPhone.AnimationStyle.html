Object Titanium.UI.iPhone.AnimationStyle

  • iPhone(0.9+), iPad(0.9+)

A set of constants for the animation styles used for view transitions.

Methods
  • addEventListener (name, callback)
  • fireEvent (name, event)
  • removeEventListener (name, callback)
Properties
CURL_DOWN [Number] Curl downwards during a transition animation.
CURL_UP [Number] Curl upwards during a transition animation.
FLIP_FROM_LEFT [Number] Flip from left to right during a transition animation.
FLIP_FROM_RIGHT [Number] Flip from right to left during a transition animation.
NONE [Number] No animation.
Methods
addEventListener (name, callback) : void

Adds the specified callback as an event listener for the named event.

Arguments

name [String] Name of the event.
callback [Callback<Object>] Callback function to invoke when the event is fired.
fireEvent (name, event) : void

Fires a synthesized event to any registered listeners.

Arguments

name [String] Name of the event.
event [Dictionary] A dictionary of keys and values to add to the Titanium.Event object sent to the listeners.
removeEventListener (name, callback) : void

Removes the specified callback as an event listener for the named event.

Arguments

name [String] Name of the event.
callback [Callback<Object>] Callback function to remove. Must be the same function passed to addEventListener.
