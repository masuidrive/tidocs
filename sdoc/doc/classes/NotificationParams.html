Object NotificationParams

  • iPhone(1.7+), iPad(1.7+)

Dictionary object of parameters used to create a notification using Titanium.App.iOS.scheduleLocalNotification.

Properties
alertAction [String] Alert button text ('View', by default) or slider text ('slide to unlock...', by default) to display.
alertBody [String] Alert message to display.
alertLaunchImage [String] Image to display instead of Default.png when launching the application.
badge [Number] Application badge value.
date [Date] Date and time for the notification to occur.
repeat [String] Interval to repeat the notification. One of weekly, daily, yearly, monthly.
sound [String] Path to the sound file to play when notification occurs, relative to the Resources folder.
timezone [String] Timezone of the date configured for the notification. If not set, the system timezone is used.
userInfo [Dictionary] Data to pass to the application with the notification event.
