Module Titanium.XML

  • Android(0.9+), iPhone(0.9+), iPad(0.9+), Mobile Web(1.8+)

The top level XML module. The XML module is used for parsing and processing XML-based content.

The API for this module is based on the W3C DOM specification.

Android and iOS implement the DOM Level 2 specification with some non-standard extensions, which are documented in the appropriate places.

Both iOS and Android lack DTD support.

Mobile Web directly exposes the DOM Level 3 implementation in the browser. New features introduced in DOM Level 3 are not documented here for brevity. For information on these features, please use the W3C documentation.

Methods
addEventListener (name, callback) : void

Adds the specified callback as an event listener for the named event.

Arguments

name [String] Name of the event.
callback [Callback<Object>] Callback function to invoke when the event is fired.
applyProperties (props) : void

Applies the properties to the proxy.

Arguments

props [Dictionary] A dictionary of properties to apply.
fireEvent (name, event) : void

Fires a synthesized event to any registered listeners.

Arguments

name [String] Name of the event.
event [Dictionary] A dictionary of keys and values to add to the Titanium.Event object sent to the listeners.
parseString (xml) : void

Parses an XML string into a Titanium.XML.Document object.

Arguments

xml [String] XML content as a string.
removeEventListener (name, callback) : void

Removes the specified callback as an event listener for the named event.

Arguments

name [String] Name of the event.
callback [Callback<Object>] Callback function to remove. Must be the same function passed to addEventListener.
serializeToString (node) : void

Serializes a Node object into a string.

Arguments

node [Titanium.XML.Node] XML Node to serialize.
