Object MusicLibraryOptionsType

  • iPhone(0.8+), iPad(0.8+)

Simple object for specifying options to openMusicLibrary.

Properties
allowMultipleSelections [Boolean] Set to true to allow the user to select multiple items from the library.
animated [Boolean] Boolean if the dialog should be animated when showing and hiding.
autohide [Boolean] Specifies that the library should be hidden automatically after media selection is completed.
cancel [Callback<Object>] Function to call if the user presses the cancel button.
error [Callback<Object>] Function to call upon receiving an error.
mediaTypes [NumberArray<Number>] An array of media type constants defining selectable media.
success [Callback<MusicLibraryResponseType>] Function to call when the music library selection is made.
