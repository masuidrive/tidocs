Object MediaItemType

  • iPhone(1.5+), iPad(1.5+)

An object representing an entry in the iPod's music library.

Properties
albumArtist [String] The album artist ID3 tag contents.
albumTitle [String] The album title ID3 tag contents.
albumTrackCount [Number] The total number of tracks on the album.
albumTrackNumber [Number] The track number of the album.
artist [String] The artist ID3 tag contents.
composer [String] The composer ID3 tag contents.
discCount [Number] The total number of discs comprising the album.
discNumber [Number] The disc number of the album.
genre [String] The genre ID3 tag contents.
isCompilation [Boolean] Whether or not the media is part of a compilation album.
lyrics [String] The lyrics ID3 tag contents.
mediaType [Number] The type of the media. One of the MUSIC_MEDIA_TYPE_* constants on Titanium.Media.
playCount [Number] The total number of plays.
playbackDuration [Number] The playback length.
podcastTitle [String] The podcast title. Only valid for media of type MUSIC_MEDIA_TYPE_PODCAST.
rating [Number] The user rating.
skipCount [Number] The total number of skips.
title [String] The title ID3 tag contents.
