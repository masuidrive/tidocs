Object Dimension

  • Android(2.0.0+), iPhone(2.0.0+), iPad(2.0.0+), Mobile Web(2.0.0+)

A simple object consisting of the position and size measurements.

When a dimension is returned by the rect property, x represents the view's left position, and y represents the view's top position. The size property returns a dimension object with x and y both set to 0.

Properties
height [Number] The height measurement.
width [Number] The width measurement.
x [Number] The x-axis coordinate of the position.
y [Number] The y-axis coordinate of the position.
