Object Titanium.Android.Calendar.Alert

  • Android(1.5+)

An object that represents a single alert for an event in an Android calendar.

Properties
alarmTime [Date] Date/time at which this alert alarm is set to trigger.
begin [Date] Start date/time for the corresponding event.
end [Date] End date/time for the corresponding event.
eventId [Number] Identifier of the event for which this alert is set.
id [String] Identifier of this alert.
minutes [Number] Reminder notice period in minutes, that determines how long prior to the event this alert should trigger.
state [Number] The current state of the alert.
Methods
addEventListener (name, callback) : void

Adds the specified callback as an event listener for the named event.

Arguments

name [String] Name of the event.
callback [Callback<Object>] Callback function to invoke when the event is fired.
fireEvent (name, event) : void

Fires a synthesized event to any registered listeners.

Arguments

name [String] Name of the event.
event [Dictionary] A dictionary of keys and values to add to the Titanium.Event object sent to the listeners.
