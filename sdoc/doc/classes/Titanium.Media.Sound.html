Object Titanium.Media.Sound

  • Android(0.8+), iPhone(0.8+), iPad(0.8+)

An object for playing basic audio resources.

The Sound object loads the entire media resource in memory before playing. If you need to support streaming, use the AudioPlayer API.

Use the Titanium.Media.createSound method to create a Sound object.

Properties
allowBackground [Boolean] Determines whether the audio should continue playing even when its activity is paused.
duration [Number] Duration of the audio resource.
looping [Boolean] Determines whether the audio should loop upon completion.
paused [Boolean] Indicates if the audio is paused.
playing [Boolean] Indicates if the audio is playing.
time [Number] Current playback position of the audio.
url [String] URL identifying the audio resource.
volume [Number] Volume of the audio, from 0.0 (muted) to 1.0 (loudest).
Events
complete Fired when the audio has finished playing.
error Fired when an error occurs while playing the audio.
interrupted Fired when audio playback is interrupted by the device.
resume Fired when audio playback is resumed after an interruption.
Code Examples
Simple Example

Simple example of playing a WAVE file from the Resources directory.

var player = Ti.Media.createSound({url:"sound.wav"});
player.play();
