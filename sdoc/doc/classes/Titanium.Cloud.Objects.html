Module Titanium.Cloud.Objects

  • Android(0.8+), iPhone(0.8+), iPad(0.8+)

Provides methods for accessing ACS custom objects.

  • create (parameters, callback)
  • query (parameters, callback)
  • remove (parameters, callback)
  • show (parameters, callback)
  • update (parameters, callback)
Code Examples
Create Custom Object

This example creates a new custom object and checks the response.

Cloud.Objects.create({
    classname: 'cars',
    fields: {
        make: 'nissan',
        color: 'blue',
        year: 2005
    }
}, function (e) {
    if (e.success) {
        var car = e.cars[0];
        alert('Success:\\n' +
            'id: ' + car.id + '\\n' +
            'make: ' + car.make + '\\n' +
            'color: ' + car.color + '\\n' +
            'year: ' + car.year + '\\n' +
            'created_at: ' + car.created_at);
    } else {
        alert('Error:\\n' +
            ((e.error && e.message) || JSON.stringify(e)));
    }
});

Query for Custom Objects

This example requests information about custom objects and checks the response.

    classname: 'cars',
    page: 1,
    per_page: 10,
    where: {
        color: 'blue'
    }
}, function (e) {
    if (e.success) {
        alert('Success:\\n' +
            'Count: ' + e.cars.length);

                'color: ' + car.color + '\\n' +
                'year: ' + car.year + '\\n' +
                'created_at: ' + car.created_at);
        }
    } else {
        alert('Error:\\n' +
            ((e.error && e.message) || JSON.stringify(e)));
    }
});
