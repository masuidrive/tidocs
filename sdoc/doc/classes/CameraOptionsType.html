Object CameraOptionsType

  • Android(0.8+), iPhone(0.8+), iPad(0.8+), Mobile Web(1.8+)

Simple object for specifying options to showCamera.

Properties
allowEditing [Boolean] Specifies if the media should be editable after capture/selection.
animated [Boolean] Specifies if the dialog should be animated upon showing and hiding.
arrowDirection [Number] Controls the type of arrow and position of the popover.
autohide [Boolean] Specifies if the camera should be hidden automatically after the media capture is completed.
cancel [Callback<Object>] Function to call if the user presses the cancel button.
error [Callback<Object>] Function to call upon receiving an error.
inPopOver [Boolean] Show the camera in a popover.
mediaTypes [Array<String>] Array of media type constants to allow: MEDIA_TYPE_PHOTO or MEDIA_TYPE_VIDEO.
overlay [Titanium.UI.View] View to added as an overlay to the camera UI (on top).
popoverView [Titanium.UI.View] View to position the camera or photo gallery popover on top of.
saveToPhotoGallery [Boolean] Specifies if the media should be saved to the photo gallery upon successful capture.
showControls [Boolean] Indicates if the built-in camera controls should be displayed.
success [Callback<CameraMediaItemType>] Function to call when the camera is closed after a successful capture/selection.
transform [Titanium.UI.2DMatrix] Transformation matrix to apply to the camera or photogallery view.
videoMaximumDuration [Number] Maximum duration (in milliseconds) to allow video capture before completing.
videoQuality [Number] Constant to indicate the video quality during capture.
