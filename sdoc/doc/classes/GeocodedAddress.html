Object GeocodedAddress

  • Android(0.8+), iPhone(0.8+), iPad(0.8+), Mobile Web(1.8+)

Simple object representing a place, returned in the callback from the reverseGeocoder method.

Properties
address [String] Full address.
city [String] City name.
country [String] Country name.
countryCode [String] Country code. On iOS, use country_code.
country_code [String] Country code. Same as country_code.
displayAddress [String] Display address. Identical to address.
latitude [String] Latitude of the geocoded point.
longitude [String] Longitude of the geocoded point.
postalCode [String] Postal code. On iOS, use zipcode.
region1 [String] First line of region.
region2 [String] Not used.
street [String] Street name, without street address.
street1 [String] Street name.
zipcode [String] Postal code. On Android, use postalCode.
