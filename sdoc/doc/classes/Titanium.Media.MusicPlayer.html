Object Titanium.Media.MusicPlayer

  • iPhone(1.4.0+), iPad(1.4.0+)

This object represents a music controller.

A MusicPlayer lets you manage and playback a queue of media Item objects.

Use the Titanium.Media.createMusicPlayer method to create a music palyer.

Use the Media module's openMusicLibrary and queryMusicLibrary methods to access the media items in the device's media library.

Properties
currentPlaybackTime [Number] Current point in song playback, in seconds.
nowPlaying [Titanium.Media.Item] An Item object representing the currently playing media item.
playbackState [Number] Playback state: one of MUSIC_PLAYER_STATE_STOPPED, MUSIC_PLAYER_STATE_PLAYING, MUSIC_PLAYER_STATE_PAUSED, MUSIC_PLAYER_STATE_INTERRUPTED, MUSIC_PLAYER_STATE_SEEK_FORWARD, or MUSIC_PLAYER_STATE_SEEK_BACKWARD.
repeatMode [Number] Repeat setting: one of MUSIC_PLAYER_REPEAT_DEFAULT, MUSIC_PLAYER_REPEAT_NONE, MUSIC_PLAYER_REPEAT_ONE, or MUSIC_PLAYER_REPEAT_ALL.
shuffleMode [Number] Shuffle setting: one of MUSIC_PLAYER_SHUFFLE_DEFAULT, MUSIC_PLAYER_SHUFFLE_NONE, MUSIC_PLAYER_SHUFFLE_SONGS, or MUSIC_PLAYER_SHUFFLE_ALBUMS.
volume [Number] Volume level for the music player, from 0.0 (muted) to 1.0 (loudest).
