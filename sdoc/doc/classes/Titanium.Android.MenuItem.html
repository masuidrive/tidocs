Object Titanium.Android.MenuItem

  • Android(1.5+)

The Titanium binding of an Android menu item.

A menu item for an Android Menu. Use Menu.add to create a new menu item.

See also: MenuItem in the Android Developer Reference.

Properties
checkable [Boolean] Determines if the item can be checked.
checked [Boolean] Determines if the item is checked.
enabled [Boolean] Determines if the item is enabled.
groupId [Number] Group ID for this item.
icon [NumberString] Icon to display for the this menu item.
itemId [Number] Item ID for this item.
order [Number] Integer used for controlling the category and sort order for menu items.
title [String] Title of the item.
titleCondensed [String] Shortened version of the item's title.
visible [Boolean] Determines whether the menu item is visible.
Events
click Fired when the user clicks the menu item.
