Module Titanium.App

  • Android(0.1+), iPhone(0.1+), iPad(0.1+), Mobile Web(1.8+)

The top level App module. The App module is mainly used for accessing information about the application at runtime.

The App module exposes a number of properties set in the tiapp.xml file.

Three of these properties, the application name, ID, and URL, must be specified when the application is created.

While most values may be changed by editing the tiapp.xml file after creating the project, the GUID is automatically generated and should not be changed.

Properties
analytics [Boolean] Indicates whether Analytics is enabled, determined by tiapp.xml.
copyright [String] Application copyright statement, determined by tiapp.xml.
deployType [String] Build type that reflects how the application was packaged; either test, development or production.
description [String] Application description, determined by tiapp.xml.
guid [String] Application globally-unique ID, determined by tiapp.xml.
id [String] Application ID, from tiapp.xml.
idleTimerDisabled [Boolean] Determines whether the screen is locked when the device is idle.
installId [String] The install ID for this application.
keyboardVisible [Boolean] Indicates whether or not the soft keyboard is visible.
name [String] Application name, determined by tiapp.xml.
proximityDetection [Boolean] Determines whether proximity detection is enabled.
proximityState [Boolean] Indicates the state of the device's proximity sensor, according to the Titanium.App.proximity event.
publisher [String] Application publisher, from tiapp.xml.
sessionId [String] Unique session identifier for the current continuous run of the application.
url [String] Application URL, from tiapp.xml.
version [String] Application version, from tiapp.xml.
Events
pause Fired when the application transitions to the background on a multitasked system.
proximity Fired when the proximity sensor changes state.
resume Fired when the application returns to the foreground on a multitasked system.
resumed Fired when the application returns to the foreground.
