Object Titanium.Media.AudioRecorder

  • iPhone(0.9+), iPad(0.9+)

An audio recorder object used for recording audio from the device microphone.

Use the Titanium.Media.createAudioRecorder method to create an audio recorder.

Properties
compression [Number] Audio compression to be used for the recording.
format [Number] Audio format to be used for the recording.
paused [Boolean] Indicates if the audio recorder is paused.
recording [Boolean] Indicates if the audio recorder is recording.
stopped [Boolean] Indicates if the audio recorder is stopped.
