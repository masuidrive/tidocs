Module Titanium.UI.iPhone

  • iPhone(0.5+), iPad(0.5+)

The iPhone/iPad-specific UI capabilities. All properties, methods and events in this namespace will only work on Apple iOS systems.

Properties
MODAL_PRESENTATION_CURRENT_CONTEXT [Number] View presented with the same style as its parent window.
MODAL_PRESENTATION_FORMSHEET [Number] Window width and height are smaller than those of the screen and the view is centered on the screen.
MODAL_PRESENTATION_FULLSCREEN [Number] Window covers the screen.
MODAL_PRESENTATION_PAGESHEET [Number] Window height is the height of the screen and width is equal to screen width in a portrait orientation.
MODAL_TRANSITION_STYLE_COVER_VERTICAL [Number] When the window is presented, its view slides up from the bottom of the screen. On dismissal, the view slides back down (default.)
MODAL_TRANSITION_STYLE_CROSS_DISSOLVE [Number] When the window is presented, the current view fades out while the new view fades in at the same time. On dismissal, a similar type of cross-fade is used to return to the original view.
MODAL_TRANSITION_STYLE_FLIP_HORIZONTAL [Number] When the window is presented, the current view initiates a horizontal 3D flip from right-to-left, resulting in the revealing of the new view as if it were on the back of the previous view. On dismissal, the flip occurs from left-to-right, returning to the original view.
MODAL_TRANSITION_STYLE_PARTIAL_CURL [Number] When the window is presented, one corner of the current view curls up to reveal the modal view underneath. On dismissal, the curled up page unfurls itself back on top of the modal view.
appBadge [Number] Value of the badge for the application's springboard icon.
appSupportsShakeToEdit [Boolean] Determines whether the shake to edit system-wide capability is enabled.
statusBarHidden [Boolean] Determines whether the status bar is hidden.
statusBarStyle [Number] Determines the status bar color style.
Methods
addEventListener (name, callback) : void

Adds the specified callback as an event listener for the named event.

Arguments

name [String] Name of the event.
callback [Callback<Object>] Callback function to invoke when the event is fired.
createNavigationGroup (parameters) : void

Creates and returns an instance of Titanium.UI.iPhone.NavigationGroup.

Arguments

parameters [Dictionary<Titanium.UI.iPhone.NavigationGroup>] Properties to set on a new object, including any defined by Titanium.UI.iPhone.NavigationGroup except those marked not-creation or read-only.
fireEvent (name, event) : void

Fires a synthesized event to any registered listeners.

Arguments

name [String] Name of the event.
event [Dictionary] A dictionary of keys and values to add to the Titanium.Event object sent to the listeners.
