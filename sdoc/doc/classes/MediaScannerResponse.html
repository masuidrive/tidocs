Object MediaScannerResponse

  • Android(0.8+), iPhone(0.8+), iPad(0.8+), Mobile Web(1.8+)

Simple object passed to the scanMediaFiles callback.

The callback is called once per scanned file.

Properties
path [String] Path to the media file that was scanned.
uri [String] URI to the file if it was scanned and added to the media library, or null if the file was not added.
