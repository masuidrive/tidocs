Module Titanium.Filesystem

  • Android(0.1+), iPhone(0.1+), iPad(0.1+), Mobile Web(1.8+)

The top level filesystem module, used to access files and directories on the device.

Properties
MODE_APPEND [Number] Constant for append mode for file operations.
MODE_READ [Number] Constant for read mode for file operations.
MODE_WRITE [Number] Constant for write mode for file operations.
applicationCacheDirectory [String] Path to the application's internal cache directory.
applicationDataDirectory [String] Path to the application's data directory.
applicationDirectory [String] Path to the iOS application directory.
applicationSupportDirectory [String] Path to the application support directory.
externalStorageDirectory [String] Path to a directory on removable storage, such as SD card.
lineEnding [String] Platform-specific line ending constant.
resourcesDirectory [String] Path to the application's resource directory.
separator [String] Platform-specific path separator constant.
tempDirectory [String] Path for the application's temporary directory.
Methods
addEventListener (name, callback) : void

Adds the specified callback as an event listener for the named event.

Arguments

name [String] Name of the event.
callback [Callback<Object>] Callback function to invoke when the event is fired.
createTempDirectory () : void

Creates a temporary directory and returns a File object representing the new directory.

createTempFile () : void

Creates a temporary file and returns a File object representing the new file.

fireEvent (name, event) : void

Fires a synthesized event to any registered listeners.

Arguments

name [String] Name of the event.
event [Dictionary] A dictionary of keys and values to add to the Titanium.Event object sent to the listeners.
getApplicationCacheDirectory () : void

Gets the value of the applicationCacheDirectory property.

Gets the value of the applicationDataDirectory property.

Gets the value of the applicationDirectory property.

getApplicationSupportDirectory () : void

Gets the value of the applicationSupportDirectory property.

getExternalStorageDirectory () : void

Gets the value of the externalStorageDirectory property.

getFile (path) : void

Returns a File object representing the file identified by the path arguments.

Arguments

path [String] One or more path components. Path arguments are joined together using the platform specific path separator to form the full path.
getLineEnding () : void

Gets the value of the lineEnding property.

getResourcesDirectory () : void

Gets the value of the resourcesDirectory property.

getSeparator () : void

Gets the value of the separator property.

getTempDirectory () : void

Gets the value of the tempDirectory property.

isExternalStoragePresent () : void

Returns true if the device supports external storage and the external storage device is mounted.

openStream (mode, path) : void

Opens file using the Titanium.IOStream interface.

Arguments

mode [Number] Access mode. Either Titanium.Filesystem.MODE_READ, Titanium.Filesystem.MODE_WRITE, or Titanium.Filesystem.MODE_APPEND.
path [String] One or more path components. Path arguments are joined together using the platform-specific path separator to form the full path.
removeEventListener (name, callback) : void

Removes the specified callback as an event listener for the named event.

Arguments

name [String] Name of the event.
callback [Callback<Object>] Callback function to remove. Must be the same function passed to addEventListener.
