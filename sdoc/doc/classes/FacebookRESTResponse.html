Object FacebookRESTResponse

  • Android(0.8+), iPhone(0.8+), iPad(0.8+), Mobile Web(1.8+)

Argument passed to the REST API callback when a request completes (successfully or unsuccessfully).

Properties
error [String] Error message, if request failed.
method [String] REST method call you specified.
result [String] JSON response returned by Facebook.
success [Boolean] Indicates if the request succeeded.
