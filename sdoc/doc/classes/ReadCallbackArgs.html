Object ReadCallbackArgs

  • Android(1.7+), iPhone(1.7+), iPad(1.7+), Mobile Web(1.8+)

Argument passed to the read callback when an asynchronous read operation finishes.

Properties
bytesProcessed [Number] Number of bytes processed, or -1 in the event of an error or end of stream.
errorDescription [String] Text description of the error.
errorState [Number] Whether an error was encountered. Set to 1 in the case of an error, 0 otherwise.
source [Titanium.IOStream] Stream being read.
