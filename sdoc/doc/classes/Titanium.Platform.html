Module Titanium.Platform

  • Android(0.1+), iPhone(0.1+), iPad(0.1+), Mobile Web(1.8+)

The top-level Platform module. The Platform module is used to access the device's platform-related functionality.

Properties
BATTERY_STATE_CHARGING [Number] Constant to indicate that the system is plugged in and currently being charged.
BATTERY_STATE_FULL [Number] Constant to indicate that the battery is fully charged.
BATTERY_STATE_UNKNOWN [Number] Constant to indicate that the battery state is not known or monitoring is disabled.
BATTERY_STATE_UNPLUGGED [Number] Constant to indicate that the system is unplugged.
address [String] The system's WIFI IP address. No other network types are supported.
architecture [String] The system's processor architecture.
availableMemory [Number] The system's unused memory, measured in megabytes on iOS and bytes on Android.
batteryLevel [Number] The battery level in percent, accessible only when batteryMonitoring is enabled. Measured in 5% increments on iPhone/iPad.
batteryMonitoring [Boolean] Determines whether battery monitoring is enabled.
batteryState [Number] Indicates the state of the battery. Accessible only when batteryMonitoring is enabled.
displayCaps [Titanium.Platform.DisplayCaps] Returns the DisplayCaps object.
id [String] The applications's globally-unique ID (UUID).
locale [String] The system's default language.
macaddress [String] The system's network interface mac address, or app UUID.
model [String] The model of the device.
name [String] The name of the platform.
netmask [String] The system's WIFI network mask. No other network types are supported.
osname [String] The short name of the system's Operating System. For example, iPhone will return iphone, iPad will return ipad, Android will return android and Mobile Web will return mobileweb.
ostype [String] The Operating System architecture. On Android, this is 32bit.
processorCount [Number] The number of processing cores.
runtime [String] The short name of the JavaScript runtime in use.
username [String] The system name, if set. On iOS, this can be found in Settings > General > About > Name.
version [String] The system's OS version.
Events
battery Fired when the battery state changes. This is measured in 5% increments on iPhone/iPad.
Code Examples
Battery Event Example

Obtaining battery data when the battery state changes.

Titanium.Platform.addEventListener('battery', function(e){
  Ti.API.info('The battery state has changed to ' + e.state);
  Ti.API.info('The battery level is ' + e.level);
  Ti.API.info('The battery event source is ' + e.source);
  Ti.API.info('The battery event name ' + e.type);
});
