Module Titanium.Utils

  • Android(0.9+), iPhone(0.9+), iPad(0.9+), Mobile Web(1.8+)

The top-level Utils module, containing a set of JavaScript methods that are often useful when building applications.

Methods
addEventListener (name, callback) : void

Adds the specified callback as an event listener for the named event.

Arguments

name [String] Name of the event.
callback [Callback<Object>] Callback function to invoke when the event is fired.
base64decode (obj) : void

Returns the specified data decoded from Base64.

Arguments

obj [StringTitanium.Blob] Data to decode.
base64encode (obj) : void

Returns the specified data encoded to Base64.

Arguments

obj [StringTitanium.Blob] Data to encode.
fireEvent (name, event) : void

Fires a synthesized event to any registered listeners.

Arguments

name [String] Name of the event.
event [Dictionary] A dictionary of keys and values to add to the Titanium.Event object sent to the listeners.
md5HexDigest (obj) : void

Returns a MD5 digest of the specified data as a hex-based String.

Arguments

obj [StringTitanium.Blob] Data to calculate.
removeEventListener (name, callback) : void

Removes the specified callback as an event listener for the named event.

Arguments

name [String] Name of the event.
callback [Callback<Object>] Callback function to remove. Must be the same function passed to addEventListener.
sha1 (obj) : void

Returns a SHA-1 hash of the specified data as a hex-based String.

Arguments

obj [StringTitanium.Blob] Data to calculate.
sha256 (obj) : void

Returns a SHA-256 hash of the specified data as a hex-based String.

Arguments

obj [StringTitanium.Blob] Data to calculate.
