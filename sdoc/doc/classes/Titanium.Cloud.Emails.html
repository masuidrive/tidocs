Module Titanium.Cloud.Emails

  • Android(0.8+), iPhone(0.8+), iPad(0.8+)

Provides methods for accessing ACS Email service.

Methods
  • send (parameters, callback)
Code Examples
Send Email

This example sends an Email and checks the response.

Cloud.Emails.send({
    template: 'welcome',
    recipients: 'joe@company.com'
}, function (e) {
    if (e.success) {
        alert('Success');
    } else {
        alert('Error:\\n' +
            ((e.error && e.message) || JSON.stringify(e)));
    }
});
Methods
send (parameters, callback) : void

Send an email to a list of email adresses.

Arguments

parameters [Dictionary] Parameters to send in the request.
callback [Callback<CloudEmailsResponse>] Callback function to execute when the method completes.
