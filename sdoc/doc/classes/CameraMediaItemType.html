Object CameraMediaItemType

  • Android(0.8+), iPhone(0.8+), iPad(0.8+), Mobile Web(1.8+)

A media object from the camera or photo gallery.

Properties
cropRect [CropRectType] Simple object defining the user's selected crop rectangle, or null if the user has not edited the photo.
media [Titanium.Blob] The media object, as a Blob.
mediaType [String] The type of media, either MEDIA_TYPE_PHOTO or MEDIA_TYPE_VIDEO defined in Titanium.Media.
