Object ActivityResult

  • Android(0.8+), iPhone(0.8+), iPad(0.8+), Mobile Web(1.8+)

Simple object passed to the startActivityForResult callback.

Properties
intent [Titanium.Android.Intent] Intent that can contain data returned to the caller. Data can be attached to the intent as "extras").
requestCode [Number] Unique, automatically generated integer request code.
resultCode [Number] Integer result code that the started activity passed to setResult.
