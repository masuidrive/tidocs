Methods
addEventListener (name, callback) : void

Adds the specified callback as an event listener for the named event.

Arguments

name [String] Name of the event.
callback [Callback<Object>] Callback function to invoke when the event is fired.
applyProperties (props) : void

Applies the properties to the proxy.

Arguments

props [Dictionary] A dictionary of properties to apply.
fireEvent (name, event) : void

Fires a synthesized event to any registered listeners.

Arguments

name [String] Name of the event.
event [Dictionary] A dictionary of keys and values to add to the Titanium.Event object sent to the listeners.
removeEventListener (name, callback) : void

Removes the specified callback as an event listener for the named event.

Arguments

name [String] Name of the event.
callback [Callback<Object>] Callback function to remove. Must be the same function passed to addEventListener.
scanMediaFiles (paths, mimeTypes, callback) : void

Scans newly created or downloaded media files to make them available to other Android media providers, such as the Gallery.

Arguments

paths [Array<String>] Array of paths to the files you want to scan.
mimeTypes [Array<String>] Array of MIME types for the files in the paths parameter. If null, MIME type will be inferred from the file names.
callback [Callback<MediaScannerResponse>] Function to call when each file is done being scanned.
setSystemWallpaper (image, scale) : void

Set the system homescreen wallpaper.

Arguments

image [Titanium.Blob] Image to use as the wallpaper, as a Blob object.
scale [Boolean] If true, the image is scaled to fit the width of the system wallpaper. If false, the image is not scaled.
