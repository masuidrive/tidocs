Module Titanium.API

  • Android(0.1+), iPhone(0.1+), iPad(0.1+), Mobile Web(1.8+)

The top-level API module, containing methods to output messages to the system log.

Methods
addEventListener (name, callback) : void

Adds the specified callback as an event listener for the named event.

Arguments

name [String] Name of the event.
callback [Callback<Object>] Callback function to invoke when the event is fired.
debug (message) : void

Logs messages with a debug severity-level.

Arguments

message [Array<String>String] Message to log. Accepts an array on iOS only.
error (message) : void

Logs messages with an error severity-level.

Arguments

message [Array<String>String] Message to log. Accepts an array on iOS only.
fireEvent (name, event) : void

Fires a synthesized event to any registered listeners.

Arguments

name [String] Name of the event.
event [Dictionary] A dictionary of keys and values to add to the Titanium.Event object sent to the listeners.
info (message) : void

Logs messages with an info severity-level.

Arguments

message [Array<String>String] Message to log. Accepts an array on iOS only.
log (level, message) : void

Logs messages with the specified severity-level.

Arguments

level [String] Log level. One of info, warn, error, debug and trace. If the value is not recognized, an info-level message prefixed with the value is logged.
message [Array<String>String] Message to log. Accepts an array on iOS only.
removeEventListener (name, callback) : void

Removes the specified callback as an event listener for the named event.

Arguments

name [String] Name of the event.
callback [Callback<Object>] Callback function to remove. Must be the same function passed to addEventListener.
timestamp (message) : void

Logs messages with a timestamp severity-level, prefixed with a timestamp float number representing the number of seconds since January 1st, 2001.

Arguments

message [Array<String>String] Message to log. Accepts an array on iOS only.
trace (message) : void

Logs messages with a trace severity-level.

Arguments

message [Array<String>String] Message to log. Accepts an array on iOS only.
warn (message) : void

Logs messages with a warn severity-level.

Arguments

message [Array<String>String] Message to log. Accepts an array on iOS only.
