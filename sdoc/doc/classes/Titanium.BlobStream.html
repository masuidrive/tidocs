Object Titanium.BlobStream

  • Android(1.7+), iPhone(1.7+), iPad(1.7+), Mobile Web(1.8+)

Wrapper around Titanium.Blob that implements the Titanium.IOStream interface

Methods
addEventListener (name, callback) : void

Adds the specified callback as an event listener for the named event.

Arguments

name [String] Name of the event.
callback [Callback<Object>] Callback function to invoke when the event is fired.
close () : void

Closes this stream.

fireEvent (name, event) : void

Fires a synthesized event to any registered listeners.

Arguments

name [String] Name of the event.
event [Dictionary] A dictionary of keys and values to add to the Titanium.Event object sent to the listeners.
isReadable () : void

Indicates whether this stream is readable.

isWriteable () : void

Indicates whether this stream is writeable.

read (buffer, offset, length) : void

Reads data from this stream into a buffer.

Arguments

buffer [Titanium.Buffer] Buffer to read stream data into.
offset [Number] Offset into the buffer to start writing stream data. If specified, length must also be specified.
length [Number] Maximum number of bytes to read. If specified, offset must also be specified.
removeEventListener (name, callback) : void

Removes the specified callback as an event listener for the named event.

Arguments

name [String] Name of the event.
callback [Callback<Object>] Callback function to remove. Must be the same function passed to addEventListener.
write (buffer, offset, length) : void

Writes data from a buffer to this stream.

Arguments

buffer [Titanium.Buffer] Buffer to write to this stream.
offset [Number] Offset in the buffer of the first byte to write to the stream. If specified, length must also be specified.
length [Number] Maximum number of bytes to write to the stream. If specified, offset must also be specified.
