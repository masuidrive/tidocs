Module Titanium.App.iOS

  • iPhone(1.5+), iPad(1.5+)

The top-level App iOS module, available only to iOS devices, that includes the facilities to create and manage local notifications and background services.

Local notifications are a way for an application that is not running in the foreground to let users know that it has information for them. When invoked, an alert dialog is displayed containing a "Close" button to dismiss and a customizable "View" button to bring the application into the foreground. Also, they can be configured to set an application icon badge, to show the number of pending notifications, and to generate a sound.

Methods
addEventListener (name, callback) : void

Adds the specified callback as an event listener for the named event.

Arguments

name [String] Name of the event.
callback [Callback<Object>] Callback function to invoke when the event is fired.
applyProperties (props) : void

Applies the properties to the proxy.

Arguments

props [Dictionary] A dictionary of properties to apply.
cancelAllLocalNotifications () : void

Cancels all scheduled local notifications.

cancelLocalNotification (id) : void

Cancels a local notification.

Arguments

id [Number] ID of the local notification to cancel.
createLocalNotification (parameters) : void

Creates and returns an instance of Titanium.App.iOS.LocalNotification.

Arguments

parameters [Dictionary<Titanium.App.iOS.LocalNotification>] Properties to set on a new object, including any defined by Titanium.App.iOS.LocalNotification except those marked not-creation or read-only.
fireEvent (name, event) : void

Fires a synthesized event to any registered listeners.

Arguments

name [String] Name of the event.
event [Dictionary] A dictionary of keys and values to add to the Titanium.Event object sent to the listeners.
registerBackgroundService (params) : void

Registers a service to run when the application is placed in the background.

Arguments

params [Dictionary] Parameters used to create the service. Must include a url property, to specify the local JavaScript file to execute when the application is placed in the background.
removeEventListener (name, callback) : void

Removes the specified callback as an event listener for the named event.

Arguments

name [String] Name of the event.
callback [Callback<Object>] Callback function to remove. Must be the same function passed to addEventListener.
scheduleLocalNotification (params) : void

Schedule a local notification.

Arguments

params [Dictionary<NotificationParams>] Parameters used to create a notification.
