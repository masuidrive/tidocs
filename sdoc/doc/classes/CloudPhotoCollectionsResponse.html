Object CloudPhotoCollectionsResponse

  • Android(0.8+), iPhone(0.8+), iPad(0.8+), Mobile Web(1.8+)

Argument passed to the callback when a request finishes successfully or erroneously.

Properties
code [Number] Error code, if any returned.
collections [Array<Dictionary>] Set of collections objects, if any exist.
error [Boolean] Indicates whether the request failed.
message [String] Error message, if any returned.
meta [Dictionary] Meta data, if any returned.
success [Boolean] Indicates whether the request succeeded.
