Object Titanium.UI.DashboardItem

  • iPhone(1.2+), iPad(1.2+)

A dashboard item is a view that is displayed as an icon in a Titanium.UI.DashboardView.

A DashboardItem is created using the Titanium.UI.createDashboardItem method.

Properties
badge [Number] Integer value displayed in a badge.
canDelete [Boolean] Determines whether this item can be deleted when it edit mode.
image [StringTitanium.Blob] Image or path to image to display in the item by default.
selectedImage [StringTitanium.Blob] Image or path to image to display in the item as it is selected.
Events
click Fired when a click is detected against the view.
delete Fired when an item is deleted during editing mode.
move Fired when an item is moved during editing mode.
