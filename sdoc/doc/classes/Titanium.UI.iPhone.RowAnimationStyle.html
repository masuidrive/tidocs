Object Titanium.UI.iPhone.RowAnimationStyle

  • iPhone(0.9+), iPad(0.9+)

A set of constants for the Animation Styles used for transition on table view rows.

Methods
  • addEventListener (name, callback)
  • fireEvent (name, event)
  • removeEventListener (name, callback)
Properties
BOTTOM [Number] The inserted row or rows slides in from the bottom; the deleted row or rows slides out toward the bottom.
FADE [Number] The inserted or deleted row or rows fades into or out of the table view.
LEFT [Number] The inserted row or rows slides in from the left; the deleted row or rows slides out to the left.
NONE [Number] No animation is performed. The new cell value appears as if the cell had just been reloaded.
RIGHT [Number] The inserted row or rows slides in from the right; the deleted row or rows slides out to the right.
TOP [Number] The inserted row or rows slides in from the top; the deleted row or rows slides out toward the top.
Methods

Fires a synthesized event to any registered listeners.

Arguments

Removes the specified callback as an event listener for the named event.

Arguments

callback [Callback<Object>] Callback function to remove. Must be the same function passed to addEventListener.
