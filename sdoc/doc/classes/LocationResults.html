Object LocationResults

  • Android(0.8+), iPhone(0.8+), iPad(0.8+), Mobile Web(1.8+)

Argument passed to the getCurrentPosition callback.

Properties
code [Number] if success is false, the error code if available.
coords [LocationCoordinates] Location data for this update.
error [String] If success is false, a string describing the error.
provider [LocationProviderDict] Object describing the location provider generating this update.
success [Boolean] Indicates if location data was successfully retrieved.
