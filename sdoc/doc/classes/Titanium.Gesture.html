Module Titanium.Gesture

  • Android(0.8+), iPhone(0.8+), iPad(0.8+), Mobile Web(1.8+)

The Gesture module is responsible for high-level device gestures such as orientation changes and shake gestures.

Methods
  • addEventListener (name, callback)
  • fireEvent (name, event)
  • getOrientation ()
  • isLandscape ()
  • isPortrait ()
  • removeEventListener (name, callback)
Properties
landscape [Boolean] True if current window is considered landscape by the device, false otherwise.
orientation [Number] Orientation of the current window.
portrait [Boolean] True if current window is considered portrait by the device, false otherwise.
Events
orientationchange Fired when the device orientation changes.
shake Fired when the device is shaken.
Methods
addEventListener (name, callback) : void

Adds the specified callback as an event listener for the named event.

Arguments

name [String] Name of the event.
callback [Callback<Object>] Callback function to invoke when the event is fired.
fireEvent (name, event) : void

Fires a synthesized event to any registered listeners.

Arguments

name [String] Name of the event.
event [Dictionary] A dictionary of keys and values to add to the Titanium.Event object sent to the listeners.
getOrientation () : void

Gets the value of the orientation property.

isLandscape () : void

Returns true if current window is considered lanscape by the device, false otherwise.

isPortrait () : void

Returns true if current window is considered portrait by the device, false otherwise.

removeEventListener (name, callback) : void

Removes the specified callback as an event listener for the named event.

Arguments

name [String] Name of the event.
callback [Callback<Object>] Callback function to remove. Must be the same function passed to addEventListener.
Events
orientationchange

Event properties

orientation [Number]

Orientation of the current window. Value is one of the following: Titanium.UI.PORTRAIT, Titanium.UI.UPSIDE_PORTRAIT, Titanium.UI.LANDSCAPE_LEFT, Titanium.UI.LANDSCAPE_RIGHT or Titanium.UI.UNKNOWN.
source [Object]

Source object that fired the event.
type [String]

Name of the event fired.
shake

Event properties

source [Object]

Source object that fired the event.
timestamp [Number]

Time interval since previous shake event. On iOS, this value is 0 for the first shake event in a series.
type [String]

Name of the event fired.
x [Number]

Acceleration along the X axis, in Gs.
y [Number]

Acceleration along the Y axis, in Gs.
z [Number]

Acceleration along the Z axis, in Gs.
