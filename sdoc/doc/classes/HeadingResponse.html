Object HeadingResponse

  • Android(0.8+), iPhone(0.8+), iPad(0.8+), Mobile Web(1.8+)

Argument passed to the getCurrentHeading callback.

Properties
code [Number] If success is false, an error code, if available.
error [String] If success is false, includes a text description of the error.
heading [HeadingData] If success is true, the actual heading data.
success [Boolean] Indicates whether the request succeeded.
